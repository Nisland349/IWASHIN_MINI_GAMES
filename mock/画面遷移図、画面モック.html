📝 補足

各画面のサイズは 390×844px（iPhone基準）。実機では Phaser.Scale.FIT でアスペクト維持しつつスクリーンサイズに合わせて拡縮されます。
進捗保存は localStorage キー iwashin_progress に保管。
このモックを Claude Code に渡せば、各シーンファイル（TitleScene.js / ChapterSelectScene.js / ChapterIntroScene.js / ChapterClearScene.js）の実装イメージが共有できます。
