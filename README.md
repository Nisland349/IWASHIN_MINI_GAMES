# いわしん製作プロジェクト（Phaser.js版）

メイドインワリオ風 5秒ミニゲーム集

## 起動方法

### 必要なもの
- Node.js（v16以上推奨）

### 起動手順

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# ブラウザで http://localhost:8080 を開く
```

> **注意**：ES Modulesを使用しているため、`file://` で直接開くと動きません。必ずローカルサーバー経由でアクセスしてください。

## スマホで確認する方法

PCとスマホを同じWi-Fiに接続し、PCのローカルIPアドレスにアクセスします：

```
http://192.168.x.x:8080  ← PCのIPアドレス
```

## Claude Codeを使った開発

このプロジェクトはClaude Code対応です。プロジェクトルートで起動してください：

```bash
claude
```

`CLAUDE.md` に開発ルールが書かれているので、Claude Codeが自動で読み込みます。

### よく使う指示例

```
「MG04_LightDodge.jsを作って。
 仕様：上から動くライトを避けて画面下から上へ移動する。
 操作はWASDとスワイプ両対応」

「MG01_SNSFire.jsの炎上投稿数を3つから2つに減らして簡単にして」
```
